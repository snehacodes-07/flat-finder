# HomeSweetHome - Flat Finder

A web application built using Flask and MySQL to help users find rental flats easily.

## Features
- View rental properties in card format
- Add new properties
- View detailed property information
- Clean UI with images

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, Bootstrap

## How to Run
1. Install Python
2. Install required packages
3. Run `python app.py`
4. Open http://127.0.0.1:5000/
